# Install Notes

Copy these files into the GoTravel repository root:

```text
AUTHORITATIVE_SPECIFICATION.md
COMMANDS.md
CODEX.md
AGENTS.md
README.md
```

For tracker signals:

- If the repo already has `TRACKER_SIGNALS.md`, do not overwrite it.
- Review `TRACKER_SIGNALS.md.template` and merge useful content manually.

Also merge the governance entry from this bundle's `CHANGES.md` into the repository `CHANGES.md`.
